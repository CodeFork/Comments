Comments as a middleware for ASP.NET Core

How-to?

  1. Install nuget package: Comments
  2. Add app.UseComments(); in Startup.cs
  3. On page where you would like to add comments add
    <div id="comments-middleware"><div>
  4. On the bottom of the body of the page on which you would like to add comments add
    <script src="/comments-middleware/loader.js"></script>
  5. Visit GitHub for configuration, how to's and more info in general.
  6. Or try posting a comment below to see how it works
